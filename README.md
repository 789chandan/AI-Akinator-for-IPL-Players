🏏 IPL AI Akinator

An AI-powered interactive web application that intelligently guesses an IPL player by asking adaptive and probability-based questions — inspired by the popular Akinator game.

📌 Project Overview

IPL AI Akinator is a smart guessing game focused entirely on the Indian Premier League (IPL) ecosystem. The system dynamically asks intelligent questions and predicts the IPL player the user is thinking about within a limited number of questions.

The project uses probability-based reasoning, dynamic candidate filtering, and adaptive questioning instead of static hardcoded decision trees.

🚀 Features
🤖 AI-powered adaptive questioning
🎯 Probability-based player prediction
🏏 Supports Indian & Overseas IPL players
📊 Dynamic candidate elimination engine
📱 Fully responsive modern UI
🖼️ Player image display with fallback handling
🔄 Session-based gameplay
⚡ Fast and lightweight PHP + MySQL implementation
🎮 Interactive Akinator-style experience
🛠️ Tech Stack
Frontend: HTML5, CSS3
Backend: PHP
Database: MySQL
Server: XAMPP / Apache
Session Handling: PHP Sessions
